#  National Highway Incident & Safety Analytics Dashboard

##  Project Overview
An end-to-end data science project that analyzes road accident data 
in India to identify patterns, hotspots, and predict accident 
severity using Machine Learning.

---

##  Live Demo
👉 [Click here to view the dashboard](https://highwaysafetydashboard-cppeihwfrjuy2f37xquzzd.streamlit.app/)

---

##  Key Insights
- 🗺️ Identified top accident prone states in India
- 🌧️ Analyzed impact of weather conditions on accident severity
- 🕐 Found most dangerous hours and days for road accidents
- 🚗 Studied vehicle types and driver age patterns
- 🤖 Built ML model to predict accident severity

##  How to Run Locally

**1. Clone the repository:**
```bash
git clone https://github.com/Dikshakumari1110/highway_safety_dashboard.git
```

**2. Install dependencies:**
```bash
pip install -r requirements.txt
```

**3. Run the dashboard:**
```bash
cd src
streamlit run app.py
```
